# Dokku Sentinel API
This is a REST API for managing dokku-acl plugin and SSH Keys by [Dokku-API](https://github.com/JeanExtreme002/Dokku-API) at the Federal University of Bahia.

## Getting Started (quick run)
Create a `.env` from `.env.sample`, configure the variables, and execute the commands below for installing and running the application:
```
$ make install
$ make run
```

## Coding Style
Run the commands below to properly format the project's code:
```
$ make lint
$ make lint-fix
```
